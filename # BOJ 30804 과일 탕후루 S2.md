# BOJ 30804 과일 탕후루 S2

## 문제

막대의 앞쪽과 뒷쪽에서 몇 개의 과일을 빼서 두 종류 이하의 과일만 남겨라.

이때 최대 과일 개수를 구해라.

## 시간복잡도

2초, 200000

## 유형 파악

투 포인터

## 풀이

투 포인터의 시작 포인터는 선언해줘야 하지만 끝 포인터는 선언이 이미 되어있다. for문의 i다.

```cpp
for(int i=0;i<n;i++){
        if(counts[arr[i]]++ == 0){
            types++;
        }
        while(types > 2){
            if(--counts[arr[s]] == 0){
                types--;
            }
            s++;
        }
        ans = max(ans, i-s+1);
    }
```

고민했던 것은, 어떻게 범위 내의 종류를 판별하는가였는데, 내부의 과일 개수를 저장하는 것으로 가능했다. 좋은 기법인 것 같다.
